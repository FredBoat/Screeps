# FredBoat Screeps AI
This has nothing to do with FredBoat except we staffers like to take a break from coding FredBoat.

## Setup
Install dependencies
```sh
npm install --dev
```

Write the credentials to `credentials.json`:
```json
{
  "email": "someone@example.com",
  "password": "foobar"
}
```

Push to Screeps:
```sh
grunt
```

OR

```sh
grunt watch
```
